# Travel Price Tracker

Αυτόματο σύστημα παρακολούθησης τιμών πτήσεων και ξενοδοχείων. Κάθε πρωί σκανάρει τους προορισμούς, βρίσκει τις καλύτερες επιλογές βάσει κανόνων, και ανανεώνει το dashboard:

**🌐 https://pymshadow.github.io/travel-tracker/**

## Λειτουργίες

- **Δυναμικές Ημερομηνίες:** Σκανάρει πολλαπλά σετ ημερομηνιών ανά προορισμό και κρατά τον συνδυασμό με το χαμηλότερο συνολικό κόστος (πτήσεις + διαμονή). Κάθε κάρτα δείχνει καθαρά τις ημερομηνίες και τις 🌙 διανυκτερεύσεις.
- **Πόλη «Έκπληξη»:** Κάθε μέρα κληρώνεται μια πόλη από δεξαμενή (Ρώμη, Πράγα, Βουδαπέστη, Μιλάνο, Βερολίνο) — πάντα διαφορετική από τη χθεσινή, και αν δεν έχει έγκυρες πτήσεις ξανακληρώνεται άλλη.
- **🏆 Top Προσφορές:** Οι καλύτερες ολοκληρωμένες προσφορές ανά πόλη κρατιούνται έως 14 μέρες με τα links τους — δεν χάνονται όταν η Έκπληξη αλλάξει πόλη.
- **Κόστος Ζωής:** Ημερήσιο κόστος (φαγητό/μετακινήσεις/έξοδοι, χωρίς διαμονή) υπολογισμένο από τιμές Numbeo, ανά άτομο × άτομα ταξιδιού, με link στην πηγή.
- **Προειδοποίηση μη-εφικτού ταξιδιού:** Αν κανένα σετ ημερομηνιών δεν έχει πτήσεις εντός κανόνων, η κάρτα το λέει ξεκάθαρα με κόκκινο πλαίσιο.

## Κανόνες

**Πτήσεις** (αυστηροί — πτήσεις εκτός ορίων αποκλείονται):
- Μόνο απευθείας
- Αναχώρηση έως 13:00 · Επιστροφή από 16:30 και μετά (χωρίς εξαιρέσεις)
- Όχι low-cost (Ryanair/Wizz κλπ.) σε ταξίδια 4+ νυχτών
- Το ⭐ δείχνει τον καλύτερο συνδυασμό τιμής+ώρας (μπόνους Aegean/Olympic, μη δεσμευτικό)
- Στόχος: ≤125€/άτομο σύνολο εισιτηρίων (εμφανίζεται ✅/▲ στην κάρτα)

**Διαμονή (Booking.com):**
- Έως 5 χλμ από το κέντρο · βαθμολογία 8+/10
- Μόνο ολόκληρα διαμερίσματα ή ξενοδοχεία με πρωινό
- Έως 800 μ από μετρό/στάση
- Όχι hostels/φοιτητικές εστίες

Όλα τα βάρη/όρια: dict `RULES` στην αρχή του `travel_tracker.py`.

## Τεχνολογίες

- **Backend:** Python + Playwright (scraping Google Flights & Booking.com)
- **Frontend:** React + Vite + TailwindCSS
- **CI/CD:** GitHub Actions — καθημερινό τρέξιμο 10:00-11:00 ώρα Ελλάδος
- **Hosting:** GitHub Pages (branch `gh-pages`, δωρεάν)

## Εκτέλεση Τοπικά

### 1. Backend (Σάρωση Τιμών)
```bash
python travel_tracker.py                      # όλα τα ταξίδια
python travel_tracker.py --single madrid-jan  # μόνο ένα
```
Ενημερώνει `snapshots/`, `history.csv`, `top_deals.json` και αντιγράφει αυτόματα ό,τι χρειάζεται στο `dashboard/public/`.

### 2. Frontend (Dashboard)
```bash
cd dashboard
npm install
npm run dev
```
Το site θα είναι διαθέσιμο στο `http://localhost:5173`.

## Προσθήκη ταξιδιού

Πρόσθεσε entry στο `trips.json` (id, from/to με κωδικούς IATA, city για το Booking, date_pairs, adults) — το επόμενο τρέξιμο το πιάνει αυτόματα. Οι πόλεις της Έκπληξης θέλουν και entry στο `dashboard/public/cost_of_living.json` για να δείχνουν κόστος ζωής.
