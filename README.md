# ✈️ Travel Price Tracker

Σύστημα καθημερινής παρακολούθησης τιμών για πτήσεις (Google Flights) και διαμονή (Booking.com).

## Κανόνες

**Διαμονή (Booking.com)** — εμφανίζονται μόνο καταλύματα που περνούν όλα τα εξής:
- έως 5 χλμ από το κέντρο
- βαθμολογία τουλάχιστον 8/10
- μόνο ολόκληρα διαμερίσματα **ή** ξενοδοχεία με πρωινό
- έως 800 μ από μετρό ή στάση (από τα δεδομένα του Booking, αλλιώς έλεγχος σε στάσεις OpenStreetMap)

**Πτήσεις** — κάθε σκέλος βαθμολογείται: προτίμηση σε όσο πιο **πρωινή αναχώρηση** και όσο πιο
**βραδινή επιστροφή**, ποινή ανά στάση, μικρό μπόνους σε Aegean/Olympic (χωρίς να είναι δεσμευτικό).
Το ⭐ στο report δείχνει την καλύτερη επιλογή βάσει κανόνων· παρακολουθείται και η απόλυτα φθηνότερη.
Τα βάρη αλλάζουν στο dict `RULES` στην αρχή του `travel_tracker.py`.

## Πώς δουλεύει

- Κάθε μέρα στις **09:00** (ή μόλις ανοίξει το PC, αν ήταν κλειστό) τρέχει αυτόματα ο έλεγχος
  μέσω του Windows Task Scheduler (task: `Travel Price Tracker`).
- Για κάθε ταξίδι στο [trips.json](trips.json) τραβάει τιμές πτήσεων και Booking.com,
  τις γράφει στο ιστορικό και ανανεώνει το **[report.html](report.html)** — άνοιξέ το στον browser.
- Στο report βλέπεις: φθηνότερη τιμή σήμερα, μεταβολή από χθες (▼/▲), badge 🔥 όταν είναι
  η χαμηλότερη τιμή που έχει καταγραφεί, γράφημα πορείας τιμής, top πτήσεις και top καταλύματα με links.

## Προσθήκη ταξιδιού

Πρόσθεσε ένα αντικείμενο στη λίστα `trips` του `trips.json`:

```json
{
  "id": "par-oct",
  "name": "Παρίσι Οκτώβριος",
  "from": "ATH",
  "to": "CDG",
  "city": "Paris",
  "depart": "2026-10-02",
  "return": "2026-10-06",
  "adults": 2,
  "enabled": true
}
```

- `from`/`to`: κωδικοί IATA αεροδρομίων (ATH, SKG, CDG, BCN, FCO, AMS, BER, ...)
- `city`: όνομα πόλης για την αναζήτηση Booking (ό,τι θα έγραφες στο site)
- `checkin`/`checkout`: προαιρετικά, αν διαφέρουν από depart/return
- `enabled: false` για να παγώσεις ένα ταξίδι χωρίς να το σβήσεις

## Χειροκίνητος έλεγχος

```
python D:\Travel\travel_tracker.py
```

## Δημιουργία Βίντεο & Social Media Uploads

Το script `make_video.py` φτιάχνει αυτόματα κάθετα βίντεο (1080x1920) με ελεύθερα πλάνα της πόλης. Το `upload_socials.py` τα ανεβάζει αυτόματα σε Instagram και TikTok.

1. **Ρύθμιση Instagram:** Φτιάξε ένα αρχείο `.env` στον φάκελο `D:\Travel\` και βάλε μέσα:
   ```env
   INSTA_USERNAME=το_username_σου
   INSTA_PASSWORD=ο_κωδικος_σου
   ```
2. **Ρύθμιση TikTok:** Κάνε login στο TikTok από τον Chrome και χρησιμοποίησε ένα extension (π.χ. EditThisCookie) για να σώσεις τα cookies σου σε ένα αρχείο `cookies.txt` στον φάκελο `D:\Travel\`.
3. **Εκτέλεση:**
   ```
   python make_video.py --all
   python upload_socials.py
   ```

## Αρχεία

| Αρχείο | Ρόλος |
|---|---|
| `trips.json` | Τα ταξίδια που παρακολουθούνται |
| `travel_tracker.py` | Το κύριο script |
| `report.html` | Το dashboard (ανανεώνεται σε κάθε έλεγχο) |
| `history.csv` | Ιστορικό ελάχιστων τιμών ανά μέρα |
| `snapshots/` | Πλήρη αποτελέσματα κάθε μέρας (JSON) |
| `tracker.log` | Log των αυτόματων εκτελέσεων |
| `run_tracker.bat` | Wrapper για το Task Scheduler |

## Σημειώσεις

- Οι τιμές πτήσεων είναι **συνολικές για όλους τους επιβάτες** (όπως τις δείχνει το Google Flights)
  και σε EUR, ανά σκέλος (αναχώρηση + επιστροφή ξεχωριστά). Οι τιμές Booking είναι
  **σύνολο διαμονής** με φόρους για τις ημερομηνίες.
- Οι πηγές είναι ανεπίσημες (scraping) — αν κάποια μέρα σπάσει κάτι, θα φανεί σφάλμα στο report
  και στο `tracker.log`. Οι τιμές είναι ενδεικτικές· επιβεβαίωσε πριν την κράτηση.
- Απενεργοποίηση αυτόματου ελέγχου: `Unregister-ScheduledTask -TaskName "Travel Price Tracker"`
