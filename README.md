# Travel Price Tracker

Αυτόματο σύστημα παρακολούθησης τιμών πτήσεων και ξενοδοχείων. Το σύστημα αναζητά καθημερινά τις καλύτερες επιλογές για συγκεκριμένους προορισμούς και εμφανίζει τα αποτελέσματα σε ένα React dashboard.

## Λειτουργίες
- **Δυναμικές Ημερομηνίες:** Σκανάρει πολλαπλά σετ ημερομηνιών για κάθε προορισμό (π.χ. 17-22 Ιαν, 18-23 Ιαν, 19-25 Ιαν) και επιλέγει τον συνδυασμό με το χαμηλότερο συνολικό κόστος (Πτήσεις + Ξενοδοχείο).
- **Πόλη "Έκπληξη":** Κάθε μέρα επιλέγει τυχαία μια πόλη από μια προκαθορισμένη δεξαμενή (Ρώμη, Πράγα, Βουδαπέστη, Μιλάνο, Βερολίνο) και βρίσκει το πιο φθηνό ταξίδι.
- **Κόστος Ζωής:** Ενσωματωμένο εκτιμώμενο ημερήσιο κόστος ζωής (για 2 άτομα) για καλύτερο υπολογισμό του budget.
- **Αυτοματοποίηση:** Εκτελείται καθημερινά μέσω GitHub Actions και ανανεώνει το site στο Netlify.

## Τεχνολογίες
- **Backend:** Python (Playwright για web scraping)
- **Frontend:** React + Vite + TailwindCSS
- **CI/CD:** GitHub Actions (χρονικά προγραμματισμένη εκτέλεση κάθε πρωί στις 11:00 ώρα Ελλάδος)
- **Hosting:** Netlify (για το frontend)

## Εκτέλεση Τοπικά

### 1. Backend (Σάρωση Τιμών)
```bash
python travel_tracker.py
```
Αυτό θα ενημερώσει τον φάκελο `snapshots/` και το αρχείο `history.csv`. Στη συνέχεια θα πρέπει να αντιγράψετε το τελευταίο snapshot στο `dashboard/public/snapshot.json`.

### 2. Frontend (Dashboard)
```bash
cd dashboard
npm install
npm run dev
```
Το site θα είναι διαθέσιμο στο `http://localhost:5173`.
